## 🚀 推荐实战项目方向

### 入门级：AI 助手面板
- 基于 Next.js 16 + React 19
- 集成 LangChain/MCP
- 实现对话历史 + 文件上传

### 进阶级：多 Agent 协作系统
- 实现需求分析 Agent + 代码生成 Agent + 测试 Agent
- 使用 MCP 协议通信
- 支持流式输出

### 高级：AI 编程助手
- 对标 Cursor/Windsurf
- 实现上下文理解 + 代码补全 + 智能问答
- 可使用 OpenViking 做上下文管理
