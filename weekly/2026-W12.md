- [原文](https://react.dev/blog/2026/02/24/the-react-foundation)

### 2. Next.js 16.2 面向 Agent 未来
- 强化 AI coding agents 集成
- 浏览器内实验性 agent
- MCP 集成和改善的日志
- **关键：把 agents 作为一等公民对待**
- [原文](https://nextjs.org/blog/agentic-future)

### 3. Vite+ 统一工具链
- Vite+ = runtime + package manager + frontend toolchain
- 一站式 Web 开发解决方案
- Rust 实现，关注性能
- [原文](https://github.com/voidzero-dev/vite-plus)

### 4. AI Agent 生态爆发
- **Agent Harness** 概念兴起（Model + Harness）
- Skills/Instincts/Memory 成为 Agent 标配
- 上下文压缩技术成熟
- 企业级 agent 平台出现（LangChain + NVIDIA）
- [The Anatomy of an Agent Harness](https://blog.langchain.dev/the-anatomy-of-an-agent-harness/)
